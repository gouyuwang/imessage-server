# A broadcast server

## Env
* ```MASTER_HOST``` master http server host
* ```MASTER_KEY``` master http server login key
* ```MASTER_HOOK_PATH``` io-server start up hook path
* ```MASTER_AUTH_PATH``` private channels auth path
